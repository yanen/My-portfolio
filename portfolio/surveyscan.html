Desktop App Survey Scan

Survey Analysis App

An application that makes the analysis of surveys easier and more readable.

Find it on Github.

How was I involved?

I co-programmed this application with my classmate Joshua as my computing project.

Overview

Survey Scan analyses sentiments of data to help identify the respondentsâ€™ general feel of the questions

It also has other statistical functions such as the calculation of the Standard Deviation, Mean, Mode, Median and so on.

Survey Scan also automatically detects the various question types (like MCQ, Long Answer, Agree/Disagree and other questions), and presents them to you for verification to ensure maximum efficiency of the analysis.

It accepts a CSV file as input and can output a CSV file if required.

Survey Scan uses Numpy, Scipy and Pandas for the extraction and analysis of data. It uses Matplotlib for data representation and Kivy for the UI.

Development process

It first started as a simple Python program for the Swift Accelerator Programme to help the course instructors in organising exit pass survey responses and tabulating attendance.

We saw the potential of the program and decided to make an app based on it that can be used to analyse any type of survey data as our computing project.

During the period of a around two months, we familiarised ourselves with the frameworks and submitted the app in February this year.
